项目简报 (最终版)
项目名称: Mercari Shops 订单处理与面单打印工具

项目目标: 在现有网站 courier.uoworld.co.jp/ 上，创建一个集成管理界面，实现从 Mercari Shops 获取订单，并一键生成日本邮政 Click Post 所需的 CSV 文件以及供仓库和客户使用的自定义发货面单。

核心功能:

订单同步: 提供一个 UI 按钮，用于从 Mercari Shops API 同步最新的“待发货”订单。

CSV 生成: 用户勾选订单后，可生成一份严格符合“日本邮政 Click Post”格式规范的 CSV 文件，并提供下载。

面单打印: 用户勾选订单后，可生成一个用于打印的、包含完整订单信息的发货面单页面。

面单必须内容: 店铺 Logo、店铺名称、收件人完整信息（姓名、地址、电话）、Mercari 订单号、下单日期、客户留言、商品明细列表（包含商品管理编号、名称、数量、价格）。

技术栈与集成环境:

集成目标: https://github.com/mkblbj/courier-management-elite

技术分析: (在对您提供的 GitHub 仓库进行分析后) 该项目是一个基于 React 和 TypeScript 的现代前端应用，使用了 Vite 作为构建工具，并集成了 Tailwind CSS 用于样式设计。新功能将采用相同的技术栈进行开发，以确保无缝集成。

关键用户体验 (UX) 要求:

界面: 新功能应作为您现有网站的一个新页面，风格与现有网站保持一致。

错误处理: 当发生任何错误（如 API 连接失败、订单信息不全等），系统需要通过弹窗的形式向用户提供清晰的提示。

开发所需关键资料:

Mercari Shops API 密钥: 需要您在 Mercari 后台为该工具申请。

Click Post CSV 规范文件: 由您提供给开发人员。

店铺 Logo 文件: 由您提供给开发人员。
